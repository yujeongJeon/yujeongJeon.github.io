6-4. 변수 인라인하기

4. 변수 인라인하기

  • 변수명이 원래 표현식과 다를바 없을 때

절차

  1. 인라인할 표현식에 사이드이펙트가 없는지 확인
  2. 상수인지 확인하고 상수로 수정 후 테스트
  3. 변수에 값이 단 한번만 대입되는지 확인
  4. 변수를 표현식으로 교체

개요 코드

// :(
const isEnd = next > -1;
return isEnd;

// :)
return next > -1;
